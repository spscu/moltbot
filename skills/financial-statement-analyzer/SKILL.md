# 财务报表分析技能

## 概述

本技能提供系统化、专业级的财务报表分析能力，帮助用户深入理解公司财务状况、识别潜在风险、做出投资决策。

**核心能力：**
- 全面解析三大财务报表（利润表、资产负债表、现金流量表）
- 计算和分析关键财务指标
- 识别财务风险和异常信号
- 生成结构化的专业分析报告

## 分析工作流程

按以下9个阶段顺序执行分析（详见 [skill-workflow.md](references/skill-workflow.md)）：

```
Stage 0: 数据校验 → 检查财报数据完整性和有效性
Stage 1: 三表解析 → 标准化利润表、资产负债表、现金流量表
Stage 2: 趋势与对标 → 分析历史趋势，对比行业基准
Stage 3: 指标计算 → 计算五维财务指标
Stage 4: 因果分析 → 分析各指标背后的驱动因素
Stage 5: 交叉验证 → 多表数据交叉验证，发现隐藏问题
Stage 6: 风险检测 → 识别财务粉饰、流动性、债务等风险
Stage 7: 综合评估 → 汇总所有分析，形成整体判断
Stage 8: 报告生成 → 输出专业分析报告
```

## 分析框架

### 1. 盈利能力分析（利润表为主）

**目标：** 判断利润是否真实、可持续、有质量

分析要点：
- 收入结构拆解（量/价/新业务来源）
- 毛利率变化趋势（竞争力变化信号）
- 费用率结构（销售/管理/研发费用分析）
- 净利率 vs 经营现金流（利润真假判断）

### 2. 成长性分析（趋势分析）

**目标：** 判断增长是否可复制、可持续

分析要点：
- 收入/利润CAGR
- 增长来源（内生 vs 并购）
- 增长成本（是否"吃掉现金"）
- 增长与现金流同步性

### 3. 运营效率分析（资产负债表为主）

**目标：** 判断资金使用效率

分析要点：
- 应收账款周转率（是否垫资）
- 存货周转率（是否滞销）
- 总资产周转率（资本效率）
- ROE杜邦分解（利润率×周转率×杠杆）

### 4. 财务安全性分析

**目标：** 判断公司是否可能"暴雷"

分析要点：
- 负债结构（短债 vs 长债）
- 利息覆盖倍数
- 有息负债/经营现金流
- 表外风险和资产减值趋势

### 5. 现金流质量分析

**目标：** 判断公司是否是"现金机器"

分析要点：
- 经营现金流/净利润比率
- 自由现金流（FCF）
- 投资现金流去向（扩张 vs 续命）
- 融资现金流依赖度

### 6. 交叉验证（高级分析）

**目标：** 发现隐藏问题

关键模式：
- 利润涨 + 现金流不涨 → 应收问题
- 收入涨 + 存货涨 → 可能虚增
- ROE高 + 高杠杆 → 风险放大
- 毛利率升 + 周转下降 → 可持续性存疑

## 输入要求

分析需要以下财务数据（至少3年历史数据）：

**利润表数据：** 营业收入、营业成本、毛利润、销售费用、管理费用、研发费用、财务费用、营业利润、净利润

**资产负债表数据：** 总资产、总负债、股东权益、流动资产、流动负债、应收账款、存货、现金及等价物、短期借款、长期借款

**现金流量表数据：** 经营活动现金流净额、投资活动现金流净额、筹资活动现金流净额、资本支出

## 输出格式

分析报告包含以下章节（详见 [report-template.md](references/report-template.md)）：

1. **执行摘要** - 核心结论和关键理由
2. **核心逻辑链** - 公司赚钱逻辑和可持续性判断
3. **结构化分析** - 分模块详细分析
4. **风险分析** - 显性和隐性风险
5. **情景推演** - 乐观/中性/悲观情景
6. **行动建议** - 投资建议和观察指标
